Cross City Transfer Learning

 

Developed & experimented with Methods & Frameworks to Transfer the knowledge from the data-rich cities to data-scarce ones using Instance, feature & model based transfer-learning techniques, and dynamic spatial similarity and periodic temporal shifts using network analysis techniques

OVERVIEW

Urban Spatiotemporal models: The heterogeneous data generated in urban areas are ubiquitous with broad application of various positioning techniques as GPS, sensing, and other sources. The vast trove of data with spatial and temporal trajectories is useful to support much urban policies and decision-making tasks like human mobility mining, traffic prediction, etc. using predictive and prescriptive analytics. Traditional statistics based approaches does not achieve the desirable predictive capabilities performance on urban-spatiotemporal data because of the following reasons:

 

  1. Urban Spatio-temporal data are multi-modal containing taxi, bike, and private vehicle trajectories, public transit swiping data, incident data, parking events, etc. All these data points are usually highly correlated, and thus it is important to analyze the data together rather than separately.
  2. The data elements present very complex Spatio-temporal dependencies, which are not captured by shallow models. This draws the attention towards Spatio-temporal deep learning models (example: ST-ResNet, STDN,  which have the capability to collectively forecast the inflow and outflow of crowd activities spatially in a city.

 

Though the deep learning models are efficient in capturing the patterns across complex Spatio-temporal boundaries, they require a large number of data points for training. But real-world systems have data scarcity due to various reasons such as data collection mechanism, data privacy issues, resource-constrained environments and no digitalization in some scenarios. These Data chain issues lead to considerable performance degradation with the deep learning predictive models. In addition, the models are specially designed for particular data types and are difficult to be generalized other types of Spatio-temporal data.

 

This is where Transfer Learning techniques come in handy, to address the data scarcity issue via using rich Spatio-temporal data from other sources. And the need is to develop an end-to-end model to automatically perform Spatio-temporal knowledge transfer and predictive capabilities.

 

Their performance may degrade remarkably with insufficient training data. Additionally, the models are specially designed for one particular data type and are hard to be generalized to handle other types of Spatio-temporal data. I have been working to use transfer learning techniques to address the data scarcity issue via using rich Spatio-temporal data from data rich cities. However, the work needs other service data to help identify whereof a city transfer learning can be performed first, and then uses it as a constraint adding to a deep learning-based prediction model. It is important to focus on capturing the common long-term trend (i.e., periodicity) of the Spatio-temporal data from multiple cities, and then transferring it to the target city. We are building end-to-end model to automatically perform Spatio-temporal knowledge transfer and predictive learning in a unified learning framework addressing the following concerns

  1. what to transfer and how between the multimodal Spatio-temporal data needs to be determined in advance
  2. How to capture cross-domain knowledge

 

Formulation of The Problem :

Given the data-rich Spatio-temporal image/representation in time series as XS = {XS t |0 ≤ t ≤ LS} in the source domain S, and the scarce Spatio-temporal image/representation in time series as XT = {XT t |0 ≤ t ≤ LT } in the target domain T, we aim to learn a function f to predict the Spatio-temporal image of the target domain in the next time-stamp by transferring the knowledge from the source domain using a minimum error function.

 

LS and LT denote the data size of the two domains and LT << LS.

 

In order to define the function f above We have been investigating  Deep adaption networks inspired by their success in various transfer learning tasks in computer vision. Deep attentive adaption networks are helpful in performing Spatio-temporal knowledge transfer and predictive learning.  Deep adaptation network (DAN) [11] generalizes deep convolutional neural network to the domain adaptation scenario through learning transferable latent features between the source domain and target domain. We first map the raw Spatio-temporal data of two domains to a common embedding space by several ConvLSTM layers. The embedded features are then input into domain-specific 3D convolution layers to learn domain-specific features. The domain-specific features are then embedded to a reproducing kernel Hilbert space for matching the embedding distributions of the two domains. To reduce the domain discrepancy, a penalty on maximum mean discrepancies (MMD), which is widely used as a distribution distance measure, is added to each domain-specific layer for joint optimization. Finally, a global spatial attention mechanism is also designed to more broadly capture the spatial dependencies of the data in two domains
