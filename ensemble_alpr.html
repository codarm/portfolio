Ensemble booster for automatic license plate recognition

Designed a Patent Pending ensemble system to boost the performance of large-scale small-object licence plate re-identification in urban settings using representational learning and decision tree.

OVERVIEW

The problem of reading the license plate numbers from the image of a vehicle involves the following steps:

  1. Detecting/localizing the license plate in the image
  2. Determining the state code/area code of the license plate
  3. Reading the numbers on the license plates
  4. Combining the results from steps 2 and 3

 

Figure 1 shows the framework of an ALPR solution that follows the above-mentioned steps.

 

There is a slightly different problem called Automatic License Plate Re-identification, in which the list of license plate numbers is already available. Given an input image, the system has to output a license plate number from the available list, which is closest to the license plate in the image. This can be posed as an Automatic License Plate Recognition problem if a huge number of templates are generated by using different combinations of characters and digits. A solution to this problem is proposed in the work of [Gupta, Mayank & Kumar, Abhinav & Madhvanath, Sriganesh. (2019). Zero-Shot License Plate Re-Identification. 773-781. 10.1109/WACV.2019.00087]. Figure 2 shows the flow diagram of an algorithm that follows this approach. We can call this a template-based ALPR solution. Fig.2 and 3 : A template-based ALPR solution  with different frameworks one using co-attention mechanisms and the other using triplet loss

 

With novel Zero-Shot Learning frameworks for license plate re-identification developed at Xerox Research as base elements, we had to merge the complementary approaches for license plate re-identification and traditional character level OCRs to boos the performance of ALPR in Complex Urban Context scenarios.  The core idea of the novel re-identification process is to generate template signatures for retrieval purposes from a multi-hot text encoding of license plates instead of their images. The proposed method maps license plate images and their license plate numbers to a common embedding space using a Symmetric Triplet loss function so that an im- age can be queried against its text.

 

A lot of research has been done in solving these problems but the demand for high accuracy ALPR systems has kept on increasing. Since ALPR systems are used in applications like surveillance, law enforcement, etc, they are required to perform with high accuracy. Instead of using a single model or method, we aim to combine the outputs of different ALPR models in order to get the best result. This is achieved by using various attention mechanisms an ensemble method. Ensemble methods perform as well as or better than any of the individual models.

 

We made the following two contributions to this invention:

  1. A novel algorithm to combine the outputs of different ALPR systems with fine-grained attention mechanisms  in order to get the best prediction
  2. A system that uses two different ALPR methods, uses their outputs to construct the input to an ensemble model which outputs the best prediction.

 

The advantage of the system is that the algorithm is very easy to implement and it is easy to train. It is also very efficient in terms of memory requirements and processing speed.
