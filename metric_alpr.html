Metric Learning for ALPR

Human in loop machine learning for knowledge transfer of License Plate manual reviews using light-weight, few-shot metric learning techniques using Image embeddings from pre-trained LPR signature recognition models.

OVERVIEW

Data domain shift and generalization are key challenges in deep learning. In the context of License-plate re-identification, self-supervised or weakly supervised models have been leveraged to learn from large-scale datasets and have shown sample-efficient transfer to labeled tasks. However, commonly used transfer techniques, e.g., fine-tuning or distillation, do not currently support few-shot, multilabel, and continual learning setup. The license plate re-identification engine with its zero-shot learning capabilities has made great strides, However, leveraging the pre-trained models to transfer learn on domain shift data is not efficient.

 

In order to make the models more robust to domain shift, I have been working on few-shot and continual human in-loop learning systems for real-world ALPR downstream tasks. The work primarily is spread across the following

Investigating to combine representations learned from pre-trained zero-shot models with few-shot continual & metric learning techniques. The proposed models use image representations, i.e., embeddings, from the license plate-re-identification engine’s f2nn layer, discussed in the previous section of Ensemble Technique.

 

Normalize the embeddings to unit length to map them to the unit sphere. Now, these weights are either used for classification or to expand the existing pre-trained model by imprinting the weights to accommodate new classes.

 

Furthermore, if we have more mislabeled or unseen classes of ALPR data, this method can be extended by mainly two strategies: average embedding and fine-tuning.

Human in Loop Machine Learning in the context of ALPR the aim is to make these 3 key contributions.
