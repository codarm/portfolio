The Symphony of Augmented Reality and Internet of Things:

Using real time data over lay techniques based on Augmented Reality  and IoT which can assist shop floor commissioning people to perform their tasks more efficiently in an electrical power plant in less time.

OVERVIEW

Recently the electric utility industry’s use of Internet of Things applications has closely followed the arc of technology availability and is getting very popular, a visualization application is necessary for users to manage complicated networked systems. However, with general visualization techniques, which map Internet of Things (IoT) devices onto a virtual space to visualize the data, the location or the data of the wireless device in virtual space is unrelated to the position of the device in real space, so it is difficult for users to associate the devices in the virtual space with those in real space, which makes it a bit complex to map things fast and analyze a situation. It is indeed a fact that the power and automation industry has very complexly connected control systems which are huge in number and this makes it further more difficult for the users to associate the virtual data with the devices in reality. This project, a visualization platform was developed based out for IoT environments using augmented reality technology. On the other hand with the means of Augmented Reality we will be able to visualize the data or status of a device in real time via Bluetooth, and even handle the previous data from the server end. The Augmented Reality based application also foresees replacing both digital and hard cover manuals in near future with a more interactive and immersed interaction with devices.

This project is concerned with the achievement of a mobile phone as a gateway for ABB electrical equipment network. The possibility, for a user, to read data collected by a certain number of Devices or sensors through a mobile device, like a phone, is a very hot topic in the context of Wireless Sensor Networks. Since more and more are their applications in frameworks where a parameter (power consumption, temperature, humidity) has to be monitored, but it’s not possible to have a PC collecting data sent from sensors or devices. Because of this reason, great challenges for this topic are to decide which is the most suitable means for sensors, devices and a mobile phone can communicate through, how to manage the message exchanges between them, and last but not the least how to display data to the user on the mobile phone. This project and prototype implementation aims to address these challenges, by developing a mobile phone as a gateway for a Feeder protection relay REF601 and IRB 140 an automated robotic Industrial Arm both ABB products. As these devices are serial communication enabled devices, Modbus protocol over Bluetooth was used for the mobile phone to communicate with the IED (Intelligent Electronic Devices). An Augmented Reality based data Display was choose in order to display both the real time data from Bluetooth and the history from the server end. The Augmented reality based application also helps the user as a manual by giving information regarding the ports of REF601. The augmented reality based applications replaced the hard cover manuals making it more easy for the user to follow the information regarding the devices he/she is dealing with.

 

The approach followed in this work considers Bluetooth as the communication protocol between sensor and mobile phone. Furthermore, the device on which the work has evolved is an Android Smartphone, for which an application has been developed. This application is designed to exchange messages with a remote device, like a sensor or a device, and to display received data not in a conventional way, but rather following the Augmented Reality approach: overlaying data to the reality framed by the camera. In a few words, after the reception of the message from the sensor or device, the user will have to frame the device or a tag itself by the camera, and the message will appear on his Trainee smart helmet.
